tetris-solver
=============

Solves tetris-fill puzzles expressed in a simple file format.

Usage:
------

    ./solver simple.puzzle

simple.puzzle:
----------

    x = 4
    y = 4
    ln = 4

Output:
-------

    Win!
    
    Piece ln at [0,0] orientation 0
            ####
    Piece ln at [0,1] orientation 0
            ####
    Piece ln at [0,2] orientation 0
            ####
    Piece ln at [0,3] orientation 0
            ####
